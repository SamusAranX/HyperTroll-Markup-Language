# HyperTroll Markup Language
* * *

## Usage

Call hypertroll.py like you would call any other Python script.

	$ python hypertroll.py denmark.jpg clusterfuck.html
	  Parsing denmark.jpg
	  Generated clusterfuck.html
	  
Opening clusterfuck.html takes about 2 seconds in Chrome on my computer.
